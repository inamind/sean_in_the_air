## 1. 게임 컨셉

> 벤치에 묶인 빨간 풍선과 노란 연.
> 바람이 불자 줄이 끊어지며 하늘로 날아오른다.
> 바람을 타고 올라가며 장애물을 피하고,
> 반짝이는 골 문에 도착하라!

### 핵심 메카닉 한 줄 요약
> **"바람은 앞으로만 밀어내지만, 벽은 바람을 이긴다."**

### 세계관
- 배경: 공원 벤치에 묶여 있던 풍선과 연이 바람에 날려 하늘로 올라감
- 스테이지가 높아질수록 더 높은 하늘로 올라가는 구조
- 낮 하늘 → 구름 → 노을 → 밤하늘 순으로 변화

## 2. 캐릭터

### 2-1. 캐릭터 종류

| 캐릭터 | 이름 | 색상 | 설명 |
|---|---|---|---|
| :balloon: | Balloon | 빨간색 | 동그란 풍선, 웃는 얼굴 |
| :kite: | Kite | 노란색 | 마름모 연, 귀여운 얼굴 |

### 2-2. 플레이어별 캐릭터 배정

| 모드 | 1P | 2P |
|---|---|---|
| **1인 플레이** | Balloon / Kite 중 선택 | - |
| **2인 플레이** | Balloon 자동 배정 | Kite 자동 배정 |

### 2-3. 이름 표시
- 게임 시작 전 영문 이름 입력
- 캐릭터 아래에 입력한 이름 항상 표시

## 3. 조작 방법

### 3-1. PC (키보드)

| 플레이어 | 왼쪽 | 오른쪽 |
|---|---|---|
| **1P** | ← 방향키 | → 방향키 |
| **2P** | A 키 | D 키 |

### 3-2. 태블릿 (화면 버튼)

| 플레이어 | 버튼 위치 | 왼쪽 | 오른쪽 |
|---|---|---|---|
| **1P** | 화면 왼쪽 하단 | ← | → |
| **2P** | 화면 오른쪽 하단 | ← | → |

### 3-3. 조작 원리
- 바람은 **자동으로 위로** 밀어올림 (플레이어가 조작 불필요)
- 플레이어는 **좌우 이동만** 조작
- 벽 아래 막대에 걸리면 **자동으로 정지 (쉼)**
- ←→ 로 막대 밖으로 나가면 **다시 바람에 날아감**

---

## 4. 핵심 메카닉

### 4-1. 바람 시스템

```
바람 ↑↑↑↑↑↑↑↑ (항상 위로)

스테이지 1~3: 위 방향 고정
스테이지 4~5: 방향 변경 추가
```

- 바람 방향 변경 **2초 전** :warning: 경고 표시
- 경고 시 화면에 **바람 방향 화살표 크게** 표시

### 4-2. 벽 & 쉼터 시스템

```
      |  노란벽  |
    ←막대      막대→
          :balloon:
    (바람 못 들어오는 쉼터)
```

- 노란벽 양옆에 **짧은 막대** 돌출
- 캐릭터가 막대 아래로 들어오면 **자동 정지**
- ←→ 로 막대 밖으로 나가면 **다시 날아감**

### 4-3. 벽 종류

| 벽 종류 | 색상 | 모양 | 기능 |
|---|---|---|---|
| :large_yellow_circle: **노란벽** | 노란색 | 평평한 벽 + 양옆 막대 | 안전 / 쉼터 |
| :red_circle: **가시벽** | 빨간색 | 가시 시각적으로 표시 | 즉사 / 생명 1개 차감 |
| :sparkles: **골 벽** | 반짝이는 골드 + 별 뿅뿅 | 반짝임 | 클리어 / 열리며 입장 |

### 4-4. 골 벽 클리어 연출 흐름

```
골 벽 도착
  ↓
:sparkles: 별 뿅뿅 + 골드빛 폭발
  ↓
골 벽이 열림
  ↓
캐릭터가 wig 모양 구름을 덮어씀
  ↓
흔들흔들 기뻐함 :joy:
  ↓
:tada: STAGE CLEAR! 텍스트 등장
  ↓
점수 대시보드 표시
```

## 6. 생명 & 게임오버 시스템

### 6-1. 생명

- 시작 시 :heart: 5개
- 화면 좌상단에 **빨간 하트**로 표시

### 6-2. 생명 차감 조건

| 조건 | 처리 |
|---|---|
| 가시벽에 닿음 | 생명 1개 차감 + 해당 스테이지 처음으로 재시작 |
| 생명 0개 | Game Over 화면 |

### 6-3. 시간 초과

| 조건 | 처리 |
|---|---|
| 제한시간 초과 | 생명 차감 없음 + 같은 스테이지 재시작 |
| 화면 표시 | "Game Over" 작게 + "Try Again?" 크게 |

## 7. 스테이지 구성

### 7-1. 스테이지별 설정

| 스테이지 | 제한시간 | 가시벽 | 바람 방향 | 배경 |
|---|---|---|---|---|
| **Stage 1** | 90초 | 없음 | 위 고정 | 낮 하늘, 나무 보임 |
| **Stage 2** | 90초 | 조금 | 위 고정 | 구름 사이 하늘 |
| **Stage 3** | 90초 | 보통 | 위 고정 | 높은 하늘, 구름 많음 |
| **Stage 4** | 120초 | 많음 | 가끔 변경 | 저녁노을 하늘 |
| **Stage 5** | 120초 | 매우 많음 | 자주 변경 | 별이 보이는 밤하늘 |

### 7-2. 난이도 곡선

```
Stage 1 ░░░░░░░░░░  튜토리얼 / 가시 없음
Stage 2 ████░░░░░░  가시 처음 등장
Stage 3 ██████░░░░  가시 늘어남
Stage 4 ████████░░  바람 방향 변경 첫 등장
Stage 5 ██████████  최종 보스 느낌
```

## 14. 의사결정 기록 (Decision Log)

### [결정 001] 게임 제목
- **결정:** 바람이 분다
- **배경:** 바람이 불면 가벼운 것들이 날아간다는 자연현상에서 출발
- **결정자:** Sean

---

### [결정 002] 바람 방향
- **후보 A:** 앞(오른쪽)으로 고정
- **후보 B:** 위로 고정 :white_check_mark:
- **후보 C:** 처음부터 방향 변경
- **결정:** 기본은 위 고정 / 스테이지 4~5에서 방향 변경 추가
- **이유:** "하늘로 날아오른다"는 세계관과 자연스럽게 맞음

---

### [결정 003] 캐릭터 설정 변경
- **초기 아이디어:** 강아지공 / 고양이공 / 쥐공 3종 선택
- **변경 후:** 빨간 풍선(Balloon) + 노란 연(Kite)
- **이유:** 세계관 "벤치에 묶인 풍선과 연"에 맞춰 캐릭터도 통일
- **참고:** 캐릭터 이미지 Sean 제공 (분홍→빨간 풍선 / 노란 연)

---

### [결정 004] 조작 방식
- **결정:** 좌우(←→)만 조작 / 바람은 자동
- **이유:** 단순한 조작으로 누구나 쉽게 접근 가능
- **2인 PC 키 배치:** 1P = ←→ / 2P = A D (키보드 공유)

---

### [결정 005] 핵심 메카닉 — 벽에서 쉬는 방법
- **문제:** 바람이 위로 부는데 어떻게 벽에 붙어서 쉬나?
- **후보 A:** 바람이 잠깐 멈출 때 붙음
- **후보 B:** 벽에 닿으면 자동 고정, 버튼 눌러야 날아감
- **후보 C:** 노란벽 양옆 막대에 걸림 / ←→ 나가면 자동 날아감 :white_check_mark:
- **결정:** Sean 직접 고안한 **막대 쉼터 시스템** 채택
- **이유:** 물리적으로 자연스럽고 플레이어가 능동적으로 판단

---

### [결정 006] 골 지점 시각 표현
- **후보 A:** 별 뿅뿅 + 골드빛 펄스 :white_check_mark:
- **후보 B:** 무지개색 테두리 + 별 반짝
- **후보 C:** 흰빛 방사형 + 별이 돌아감
- **후보 D:** 금색 물결 + 별 스파클
- **결정:** A 선택 (Sean 선택)
- **추가:** 골 벽이 **열리는 연출** 추가 (Sean 아이디어)

## 15. 코딩 제작 순서 (예정)

```
1단계  → 시작화면 + 인원선택 + 이름입력 + 캐릭터선택
2단계  → 배경 (스테이지별 하늘) + 캐릭터 표시
3단계  → 바람 물리 + ←→ 이동 구현
4단계  → 노란벽 + 막대 쉼터 시스템
5단계  → 가시벽 + 생명 차감
6단계  → 골 벽 열리는 연출 + wig 구름 애니메이션
7단계  → 동전 + 점수 시스템
8단계  → 타이머 + 시간초과 처리
9단계  → 스테이지 클리어 대시보드
10단계 → 2인 플레이 + 카메라 시스템
11단계 → 레트로 BGM + 효과음 (Web Audio API)
12단계 → 엔딩 화면
13단계 → 전체 테스트 + 밸런스 조정
```
